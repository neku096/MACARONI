# AGENTS.md

このプロジェクトでは、既存のWebサイト構造を壊さず、BOOTH誘導とSEO改善を安全に進める。

## 最優先

- 変更前に関連ファイルを読む。
- 既存のレイアウト、年齢確認、SEO、ライトボックス、関連商品、フィルター構造を維持する。
- 不明な対応キャラや仕様を推測でタグ付けしない。
- 大きな作り直しや不要な削除をしない。
- 誇大表現やBOOTH本文にない断定を避ける。

## BOOTH作品ページの注意

- シリーズタグとキャラサブタグは別物として扱う。
- 汎用モーション、汎用マテリアルはキャラサブタグに含めない。
- フェラモーション、一人エッチシリーズは、BOOTHまたはユーザー提示の対応アバター一覧で確認できる場合だけキャラサブタグに含める。
- キャラサブタグを付ける前に、商品ごとの対応アバター一覧を確認する。
- 画像ギャラリーのaltやサムネ名だけで対応キャラを確定しない。正式な対応一覧がある場合はそちらを優先する。

## 今回の失敗からのチェック

- ショコラはフェラモーション対応ではないため、フェラモーションのキャラサブタグに含めない。
- 一人エッチ vol.2 / vol.3 / vol.4 は正式な対応アバター一覧に基づいてタグ付けする。
- 一人エッチ vol.2 / vol.3 / vol.4 は、りりかをキャラサブタグに含めない。
- 一人エッチ vol.2 / vol.3 は、エク、ルミナ、ショコラを含める。
- 一人エッチ vol.4 は、ルミナ、エク、イチゴ、ミルティナ、しなの、マヌカ、ルルネ、しお、ショコラ、ミルフィ、萌、セレスティアを含める。
- サブタグUIを追加・変更したら、HTMLだけでなく `styles.css` 側に対応クラスがあるか確認する。
- `product-tag-chip` を使う場合は、見た目のフレーム、ホバー、余白が既存UIと揃っているか確認する。
- サブタグの並びは、日本語ページは五十音順、英語ページはABC順にする。

## 変更後の確認

- 作業完了前にこの `AGENTS.md` のミス防止メモを読み返し、同じミスが再発していないか確認する。
- 対象HTMLの構造が壊れていないこと。
- `product-gallery-data` がある場合は `JSON.parse` できること。
- 参照画像パスが存在すること。
- `script.js` の構文チェックを通すこと。
- キャラサブタグで、該当しない商品が混ざらないこと。
